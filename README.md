# Boarding Project - Product

A modern boarding space finder application with owner and student management.

## Project Structure

- `Backend/` - Express.js backend server
- `Frontend/` - Next.js frontend application

## Setup

See individual README files in Backend and Frontend directories for setup instructions.

## Environment Variables

Make sure to set up `.env` files in both Backend and Frontend directories. See `Backend/ENV_SETUP.md` for details.
